** Requirements:
1. Python 3
2. MNIST data set

** Instructions:
1. Download the source code files - LoadMNIST.py, PA2.py and 3d.py, and place them in a folder such that the same folder has all the MNIST data files in unzipped format.
2. Run 3d.py for gradient calculation part of question 3, part d
3. Run PA2.py for all other parts. Scroll down to "main" of this python file to modify parameters such as momentum, layer configuration, learning rate etc.
